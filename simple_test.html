Простой тест Mail7 Email Scanner

1. Email-адреса в обычном тексте:

Свяжитесь с нами: test@example.com

Поддержка: support@test.org

Информация: info@company.net

2. Email-адреса в ссылках:

Написать нам | Администратор

3. Email-адреса в атрибутах:

Элемент с data-email

4. Email-адреса в title:

Элемент с title
Информация

5. Сложные email-адреса:

С подчеркиванием: user_name@example.com

С точками: user.name@test.org

С дефисами: user-name@company.net

С цифрами: user123@example.org

С плюсом: user+tag@test.com

6. Невалидные email-адреса (для проверки фильтрации):

Неполный: user@

Без домена: user@.com

Неправильный формат: user.example.com

7. Дублирующиеся email-адреса:

Первый: duplicate@example.com

Второй: duplicate@example.com

Третий: DUPLICATE@EXAMPLE.COM

Инструкции по тестированию:

  1. Установите плагин Mail7 Email Scanner
  2. Откройте эту страницу
  3. Нажмите на иконку плагина
  4. Нажмите "Сканировать страницу"
  5. Должно быть найдено около 10+ уникальных email-адресов

Ожидаемые результаты:

  • Должно быть найдено 10+ уникальных email-адресов
  • Дубликаты должны быть удалены
  • Невалидные адреса должны быть отфильтрованы
  • Email-адреса должны быть приведены к нижнему регистру
